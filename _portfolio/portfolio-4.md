This project was done in a collaboration with my classmate, Kimberly Liu. The goal of this project was to use the Tensor Basis Neural Network (TBNN) to predict the Reynolds stress tensor in canonical flow configurations. We tested our model on turbulent channel and turbulent Couette flow using data from the UT Austin Oden Institute. For full details of the project, see the full report linked [here](htpps://cagonzal.github.io/files/CS230_Final_Report.pdf)
