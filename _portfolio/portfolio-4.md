This project was done in a collaboration with my classmate, Kimberly Liu. The goal of this project was to use the Tensor Basis Neural Network (TBNN) to predict the Reynolds stress tensor in canonical flow configurations. We tested our model on turbulent channel and turbulent Couette flow using data from the UT Austin Oden Institute. Pictured below is a comparison between the DNS and TBNN values of the b<sub>12</sub> component of the Reynolds anisotropy tensor as a function of wall-normal height for channel flow at Re<sub>$\tau$</sub> = 550.
